🔥ÉTAPE 1 : Création du compte Claude (Gratuit)

1.1 Accéder au site Claude

  1. Ouvrez votre navigateur web
  2. Rendez-vous sur : https://claude.ai/
  3. Cliquez sur "Sign Up" ou "Get Started"

1.2 Créer votre compte

  1. Saisissez votre email professionnel ou personnel
  2. Créez un mot de passe sécurisé (8+ caractères, majuscules, chiffres)
  3. Acceptez les conditions d'utilisation
  4. Cliquez sur "Create Account"

1.3 Vérification email

  1. Consultez votre boîte email
  2. Cliquez sur le lien de vérification reçu d'Anthropic
  3. Confirmez votre compte - Vous serez redirigé vers Claude

✅Vérification Étape 1

  • Compte Claude créé et vérifié
  • Connexion réussie à https://claude.ai/
  • Interface de chat Claude accessible

💳ÉTAPE 2 : Ajout de crédits (20$ USD)

2.1 Accéder à la console de facturation

  1. Depuis Claude.ai, cliquez sur votre profil (coin supérieur droit)
  2. Sélectionnez "Console" ou allez directement à :
https://console.anthropic.com/settings/billing

2.2 Configuration du paiement

  1. Dans l'onglet "Billing", cliquez sur "Add Payment Method"
  2. Saisissez vos informations de carte de crédit :
    • Numéro de carte
    • Date d'expiration (MM/AA)
    • Code CVC
    • Nom sur la carte
    • Adresse de facturation

2.3 Ajout des crédits

  1. Cliquez sur "Add Credits"
  2. Sélectionnez 20$ USD (montant recommandé pour commencer)
  3. Confirmez la transaction
  4. Vérifiez que le solde apparaît dans votre compte

💡Informations importantes

  • Coût d'utilisation Claude 4 : ~0.015$ par 1000 tokens
  • 20$ USD = ~1,300,000 tokens (environ 500-1000 conversations expertes)
  • Pas d'abonnement : Paiement à l'usage uniquement
  • Alertes configurables : Définissez des limites de dépense

✅Vérification Étape 2

  • Carte de crédit ajoutée avec succès
  • Solde de 20$ USD visible dans le compte
  • Reçu de transaction reçu par email

🔑ÉTAPE 3 : Création de la clé API

3.1 Accéder aux paramètres des clés

  1. Dans la console Anthropic, naviguez vers :
https://console.anthropic.com/settings/keys

Ou cliquez sur "API Keys" dans le menu de gauche

3.2 Générer une nouvelle clé

  1. Cliquez sur "Create Key" ou "+ New Key"
  2. Nommez votre clé : EXPERTS_IA_Production
  3. Sélectionnez les permissions :
    • ✅ Messages (requis)
    • ✅ Web Search (recommandé)
    • ✅ Tool Use (recommandé)
  4. Cliquez sur "Create Key"

3.3 Copier et sécuriser la clé

⚠️IMPORTANT

La clé ne s'affichera qu'une seule fois !

  1. Copiez immédiatement la clé complète :
ANTHROPIC_API_KEY = sk-ant-api03-[VOTRE_CLÉ_UNIQUE]
  1. Sauvegardez-la dans un endroit sécurisé (gestionnaire de mots de passe)

🔒Sécurité des clés API

  • Ne partagez JAMAIS votre clé API
  • Ne la commitez pas dans un dépôt public
  • Régénérez-la si elle est compromise
  • Surveillez l'usage dans la console

✅Vérification Étape 3

  • Clé API créée avec nom descriptif
  • Clé copiée et sauvegardée en sécurité
  • Format correct : sk-ant-api03-...

🤗ÉTAPE 4 : Création du compte Hugging Face

4.1 Inscription sur Hugging Face

  1. Rendez-vous sur : https://huggingface.co/
  2. Cliquez sur "Sign Up" (coin supérieur droit)

4.2 Création du compte

  1. Choisissez votre nom d'utilisateur : votre-nom-expert-ia
  2. Saisissez votre email (peut-être le même que Claude)
  3. Créez un mot de passe sécurisé
  4. Acceptez les conditions d'utilisation
  5. Cliquez sur "Sign Up"

4.3 Vérification et profil

  1. Vérifiez votre email avec le lien reçu
  2. Complétez votre profil :
    • Photo de profil (optionnel)
    • Bio courte (ex: "Expert IA Construction Québec")
    • Liens vers vos réseaux (optionnel)

💡Conseils Hugging Face

  • Nom d'utilisateur : Choisissez-le bien, difficile à changer
  • Profil public : Soignez-le, il représente vos projets IA
  • Communauté active : Explorez les modèles et datasets

✅Vérification Étape 4

  • Compte Hugging Face créé et vérifié
  • Nom d'utilisateur choisi et disponible
  • Profil complété (au minimum bio)

5.2 Dupliquer l'espace choisi

  1. Rendez-vous sur l'URL de l'application choisie
  2. Connectez-vous à votre compte Hugging Face
  3. Cliquez sur "Duplicate this Space" (bouton en haut à droite)
  4. Configurez votre nouvel espace :
    • Owner : Sélectionnez votre compte
    • Space name : Utilisez le nom suggéré ou personnalisez
    • Visibility : Public (recommandé) ou Private
    • Hardware : CPU basic (gratuit, suffisant pour commencer)
  5. Options avancées (optionnel) :
    • Description : Adaptez selon l'application choisie
    • License : Apache-2.0 (recommandé)
    • SDK : Streamlit (déjà configuré)
  6. Cliquez sur "Duplicate Space"

🔐ÉTAPE 6 : Configuration de la clé API (Secrets)

6.1 Accéder aux paramètres de l'espace

  1. Dans votre espace dupliqué, cliquez sur "Settings" (onglet en haut)
  2. Scrollez jusqu'à la section "Variables and secrets"

6.2 Ajouter la clé API comme secret

  1. Dans "Repository secrets", cliquez sur "New secret"
  2. Configurez le secret :
    • Name : ANTHROPIC_API_KEY
    • Value : Collez votre clé complète sk-ant-api03-...
  3. Cliquez sur "Add secret"

6.3 Vérification de la configuration

  1. Vérifiez que le secret apparaît dans la liste (valeur masquée)
  2. Redémarrez l'espace en cliquant sur "Factory reboot" dans Settings
  3. Attendez le redémarrage complet (2-3 minutes)

🔒Sécurité des secrets Hugging Face

  • Secrets chiffrés : Stockage sécurisé par Hugging Face
  • Invisibles : Ne s'affichent jamais dans les logs publics
  • Accès restreint : Seulement le propriétaire de l'espace
  • Révocables : Peuvent être supprimés/modifiés à tout moment

✅Vérification Étape 6

  • Secret ANTHROPIC_API_KEY ajouté avec succès
  • Valeur de la clé correctement saisie (sk-ant-api03-...)
  • Espace redémarré après ajout du secret

❌Application ne démarre pas

  • Consultez les logs dans l'onglet "Logs" de l'espace
